# @ym/GKD_subscription

---

由ym维护的GKD规则

**仅启用 `开屏广告` 一类规则，其它所有规则均需用户手动打开，您可以在规则类别里批量开启某一类规则**

在 GKD 内添加以下链接即可使用此规则

```txt
https://raw.githubusercontent.com/Ym-wei/GKD/main/dist/ym.json5
```

当前版本: v20
